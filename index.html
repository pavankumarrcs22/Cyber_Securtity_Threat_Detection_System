AI-Powered Cybersecurity Threat Detection in Automotive Systems

This project focuses on detecting cyber threats in automotive systems using AI-based models. The system analyzes real-time CAN Bus data and classifies it into different attack categories such as DoS, Fuzzy, and Impersonation. The trained model helps in identifying malicious activities to enhance the security of modern automotive networks.

Key Features:

  • Uses Machine Learning to analyze CAN Bus data.
  • Detects various types of cyber attacks in automotive systems.
  • Provides real-time attack classification and alerts.
